## Getting Started

Clone this Repository using `git clone https://github.com/Prakashh21/Netflix_UI.git` <br/>
<br/>
Change directory to the project directory using `cd Netflix_UI` <br/>
<br/>
Run command `npm i && npm start` 
<br/>
